<h1 align="center">
  vue-pdf-reader
</h1>
This code is offered as a component to render PDF with text layer in web by using Vue.js and PDF.js.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
